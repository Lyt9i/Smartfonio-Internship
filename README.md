🐭 MouseAdventure

📖 Сюжет
Маленькую мышку похитили и заперли в темной коробке.
Её цель — сбежать на свободу, но для этого нужно найти 3 кусочка сыра, разбросанные по уровню.

С каждым съеденным кусочком мышка набирается сил и увеличивает скорость — она чувствует, что свобода близка!

👾 Главный враг
Но мышка не одна в коробке...

Тень Руки — зловещий силуэт, который постоянно движется в сторону мышки.
Если тень коснётся игрока 3 раза — игра заканчивается поражением.

⚠️ В текущей версии показывается сообщение о проигрыше. В будущем рука заберёт мышку с экрана!
🎮 Геймплей
Действие			Результат
🧀 Собрать кусочек сыра		+1 к прогрессу, увеличение скорости мышки
👻 Коснуться тени		−1 жизнь (всего 3 жизни)
🏆 Собрать 3 сыра		Победа! Мышка выходит из коробки
💀 Потерять 3 жизни		Поражение — игра заканчивается

🖥️ Управление
W / A / S / D или Стрелки — движение мышки

🧩 Особенности
🧀 Динамическая скорость персонажа

👤 Интеллектуальное преследование тени

📦 Одна замкнутая сцена (коробка)

🎯 Простая, но напряжённая механика

🖐️ Атмосфера "присутствия" руки-злодея
